Backend Developer Intern / Trainee

REVLV Solutions Inc.

As a Backend Developer Intern at REVLV Solutions Inc., I developed ExpenSEEs-API, a RESTful backend for expense tracking using Spring Boot, Java, and PostgreSQL. My responsibilities included implementing user management, budget operations, expense tracking, and liquidation reports. I also integrated JWT-based authentication and Spring Security for role-based access control, configured and tested API endpoints using Postman and IntelliJ IDEA, and collaborated with frontend developers to integrate the API with the Admin Dashboard and mobile app. Finally, I deployed the backend on a DigitalOcean server, ensuring CORS configuration for seamless frontend connectivity.
